
  # Untitled

  This is a code bundle for Untitled. The original project is available at https://www.figma.com/design/ALAUAz3ezWmkeXsGB51rs1/Untitled.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Backend API base URL

  By default, frontend requests go to `http://localhost:8080`.
  To change this, set:

  `VITE_API_BASE_URL=http://your-backend-host:port`
